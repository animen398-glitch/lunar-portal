┌─────────────────────────────────────────────────────────────┐
│  ШАГ 1: ЗАПУСК САЙТА                                         │
└─────────────────────────────────────────────────────────────┘

   ВАРИАНТ А (САМЫЙ ПРОСТОЙ):
   ──────────────────────────
   1. Найдите файл "index.html"
   2. Дважды кликните на него
   3. Сайт откроется в браузере!

   ⚠️ Если переводы не работают, используйте Вариант Б


   ВАРИАНТ Б (ЧЕРЕЗ СЕРВЕР):
   ──────────────────────────
   1. Дважды кликните на файл "start.bat"
   2. Дождитесь сообщения о запуске сервера
   3. Откройте браузер
   4. Введите: http://localhost:8000


┌─────────────────────────────────────────────────────────────┐
│  ШАГ 2: ЕСЛИ НИЧЕГО НЕ РАБОТАЕТ                              │
└─────────────────────────────────────────────────────────────┘

   Установите Python (бесплатно, 2 минуты):
   
   1. Откройте: https://www.python.org/downloads/
   2. Скачайте последнюю версию Python
   3. Запустите установщик
   4. ⚠️ ВАЖНО: Отметьте галочку "Add Python to PATH"
   5. Нажмите "Install Now"
   6. После установки запустите start.bat снова


┌─────────────────────────────────────────────────────────────┐
│  ШАГ 3: ИСПОЛЬЗОВАНИЕ                                        │
└─────────────────────────────────────────────────────────────┘

   ✅ Выберите дату в правом верхнем углу
   ✅ Переключайте языки кнопками EN / RU
   ✅ Кликайте на кнопки "Day 1", "Day 2" и т.д. для просмотра
      описаний разных лунных дней
   ✅ Все данные автоматически обновляются при выборе даты


┌─────────────────────────────────────────────────────────────┐
│  ШАГ 4: ПОДКЛЮЧЕНИЕ API (опционально)                       │
└─────────────────────────────────────────────────────────────┘

   Сайт УЖЕ РАБОТАЕТ без API!
   
   Но если хотите подключить реальные данные:
   
   1. Откройте файл: services/ephemeris.js
   2. Найдите строку: swissEphemerisUrl
   3. Замените URL на адрес вашего API
   4. Сохраните файл
   5. Перезапустите сайт
