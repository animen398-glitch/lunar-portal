  КАК ЗАГРУЗИТЬ ФАЙЛЫ ПО ЧАСТЯМ (если Git не работает)
═══════════════════════════════════════════════════════════════

Если не хотите использовать Git, можно загрузить файлы по частям

───────────────────────────────────────────────────────────────

ШАГ 1: Создайте репозиторий
───────────────────────────────────────────────────────────────

1. Откройте: https://github.com/new
2. Repository name: lunar-portal
3. Visibility: Public
4. НЕ отмечайте "Add README"
5. Нажмите "Create repository"

───────────────────────────────────────────────────────────────

ШАГ 2: Загрузите основные файлы (первая партия)
───────────────────────────────────────────────────────────────

1. На странице репозитория нажмите "Add file" → "Upload files"
2. Перетащите ТОЛЬКО эти файлы:
   ✅ index.html
   ✅ styles.css
   ✅ app.js
   ✅ README.md
3. Нажмите "Commit changes"

───────────────────────────────────────────────────────────────

ШАГ 3: Создайте папку locales
───────────────────────────────────────────────────────────────

1. На странице репозитория нажмите "Add file" → "Create new file"
2. Введите: locales/en.json
3. Скопируйте содержимое файла en.json из вашей папки
4. Нажмите "Commit new file"
5. Повторите для locales/ru.json

───────────────────────────────────────────────────────────────

ШАГ 4: Создайте папку services
───────────────────────────────────────────────────────────────

1. Нажмите "Add file" → "Create new file"
2. Введите: services/ephemeris.js
3. Скопируйте содержимое файла ephemeris.js
4. Нажмите "Commit new file"
5. Повторите для services/lunarDays.js

───────────────────────────────────────────────────────────────

ШАГ 5: Включите GitHub Pages
───────────────────────────────────────────────────────────────

1. Settings → Pages
2. Source: Deploy from a branch
3. Branch: main
4. Folder: / (root)
5. Save

───────────────────────────────────────────────────────────────

⚠️ ВАЖНО: Этот способ медленный, но работает!

───────────────────────────────────────────────────────────────
