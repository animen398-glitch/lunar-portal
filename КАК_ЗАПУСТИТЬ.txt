📌 СПОСОБ 1: САМЫЙ ПРОСТОЙ (РЕКОМЕНДУЕТСЯ)
─────────────────────────────────────────────

1. Найдите файл "index.html" в этой папке
2. Дважды кликните на него
3. Сайт откроется в браузере!

✅ Готово! Всё работает.


📌 СПОСОБ 2: ЧЕРЕЗ СЕРВЕР (для полной функциональности)
────────────────────────────────────────────────────────

Если просто открыть файл не работает (переводы не загружаются):

1. Дважды кликните на файл "start.bat"
2. Дождитесь сообщения "Starting local web server..."
3. Откройте браузер
4. Введите адрес: http://localhost:8000


📌 ЕСЛИ НИЧЕГО НЕ РАБОТАЕТ
──────────────────────────

Установите Python (бесплатно):
1. Откройте https://www.python.org/downloads/
2. Скачайте и установите Python
3. При установке ОБЯЗАТЕЛЬНО отметьте "Add Python to PATH"
4. После установки запустите start.bat снова


📌 НАСТРОЙКА API (опционально)
───────────────────────────────

Сайт УЖЕ РАБОТАЕТ без API! Но если хотите подключить реальные данные:

1. Откройте файл: services/ephemeris.js
2. Найдите строку с "swissEphemerisUrl"
3. Замените URL на адрес вашего API
4. Сохраните файл


📌 ПЕРЕКЛЮЧЕНИЕ ЯЗЫКОВ
──────────────────────

В правом верхнем углу есть кнопки EN и RU
Просто кликните на нужный язык!
